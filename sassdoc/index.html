    [private] grid-calc

    @mixin grid-calc($prop, $widths) { ... }

    Description

    Calculate all widths for a call from i, push or pull

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $prop

    Can be flex-basis, left or right

    		String—none
    $widths

    List of breakpoint / width-pairs

    		List—none

    Output

    Widths and breakpoints for a mixin-call

    Throws

    • You need to pass at least one statement to push/pull

    • You cannot pass false to breakpoint #{$key}.

    Requires

    • [mixin] bp
    • [function] calculate-width
    • [function] parse-breakpoint

    Used by

    • [mixin] item
    • [mixin] push
    • [mixin] pull

    Author

    • Felix Sp√∂ttel

    @mixin grid-modifiers($modifiers) { ... }

    Description

    • center

    • space-between
    • rev
    • flip

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $modifiers

    See README

    		List—none

    Output

    Modifier-styles for a given call
